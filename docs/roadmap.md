# Roadmap

## Production Baseline

- Local-first OAuth setup.
- Read-only default scopes.
- GPS privacy controls.
- Activity, route, stream, gear and club tools.
- Daily and weekly training summaries.

## Next Improvements

- More stream analytics: decoupling, pace drift, HR drift, climb segmentation.
- Better route intelligence: elevation profile, surface hints, privacy-safe route summaries.
- Optional webhook support for low-polling sync.
- More MCP client examples and video walkthroughs.
- Evaluation fixtures for realistic endurance-coach questions.

## Explicit Non-Goals For Now

- No private/internal Strava endpoints.
- No default write/upload scopes.
- No claim of continuous raw sensor telemetry.
- No medical diagnosis or treatment advice.
